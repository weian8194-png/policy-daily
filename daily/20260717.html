🔴紧急 CBP IOR Error Code 333今日(7/16)正式部署——一年未用IOR自动标"非活跃"
7/16已部署

CBP 7/16正式在ACE生产环境部署"入境目的非活跃"状态。一年(366天)及以上无报关记录且无未结清业务的IOR号，系统自动标记为非活跃。货物放行申报触发Error Code 333，报关摘要申报触发F875。激活方式：通过报关行提交TP消息或向CBP发邮件附CBP Form 5106申请将状态从20-Inactive恢复为10-Active。根源于6/3《加强海关执法》行政令要求180天内清理IOR登记册。

🎯 3步行动清单
  1. 排查所有在途货物所用IOR最后申报日期，确认是否处于活跃状态
  2. 非活跃IOR通过报关行提交TP消息或发邮件CBP Form 5106申请激活
  3. 建立IOR使用台账，确保每个IOR至少每年有一次报关记录
